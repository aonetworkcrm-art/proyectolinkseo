## 🔧 Cómo Actualizar (Sin Romper Nada)

### Subir cambios a GitHub Pages:
```bash
# 1. Ve al proyecto
cd /c/proyecto-link-seo

# 2. Agrega tus cambios
git add .

# 3. Haz commit
git commit -m "Descripción de lo que cambiaste"

# 4. Sube a GitHub
git push
```
> ✅ GitHub Pages se actualiza solo en 1-2 minutos.

### Si quieres crear un nuevo archivo HTML:
1. Crea tu archivo `.html` en la carpeta que corresponda
2. Agrega un link en `dashboard-central.html`
3. Haz `git add`, `git commit`, `git push`

### ⚠️ Reglas para no romper nada:
- **No borres archivos** sin verificar que no hay links apuntando a ellos
- **No edites `.gitignore`** a menos que sepas lo que haces
- **No subas datos sensibles** (tokens, contraseñas, bases de datos de terceros)
- **Usa `dashboard-central.html`** como centro para agregar nuevos módulos
